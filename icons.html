Key lines

Keyline shapes are the foundation of the grid. By using these core shapes as guidelines, you can maintain consistent visual proportions across system icons.

Key shapes

FAQs

Base grid

Icon content should remain inside of the live area.

Stroke consistency

Use the same thickness across all icons to maintain consistency. Do not manipulate the thickness.

Set consistency

Use the same set of icons across all 12 icons. Eg: Do not use 2px for Pay and 1.5px for Bank transfer.

Themes

Pratima only supports stroked icons, do not manipulate the icons to fill or add gradients.

Success

Use tick and green circle. Any size, minimum 24 x 24 dp.

Failure

Use a white cross on a red circle.
